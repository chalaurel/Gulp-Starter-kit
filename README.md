# gulp-starter-kit

## Usage
```
git clone https://github.com/chalaurel/gulp-starter-kit.git
```
### go inside the repo 
```
cd gulp-starter-kit
```
###  Install all dependencies (make sure nodejs with npm is installed on your machine)

```
### Run gulp default to build 

```
gulp
```

### List of npm used

gulp-imagemin
gulp-sass
